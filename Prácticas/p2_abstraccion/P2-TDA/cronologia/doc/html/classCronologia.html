Descripción detallada

T.D.A. Cronología Una instancia c del tipo de datos abstracto Cronologia es un objeto que contiene una cronología de acontecimientos, hechos o eventos, ordenados cronológicamente por años. Cada objeto posee una secuencia de fechas históricas en la que cada una tiene los diversos hechos.
